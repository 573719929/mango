<h2><a name="" class="anchor" href="#"><span class="mini-icon mini-icon-link"></span></a>介绍</h2>
<p>mango是移动版是以webkit为内核的浏览器专门编写的javascript库,在BSD协议下开源发布。</p>

<p>mango即芒果</p>
<p>此库可以说是jquery的超精简版，去掉了所有浏览器兼容的代码，接口却基本与jquery保持一至</p>
